About the Project

This website was developed as part of the CT1452 course project at King Saud University, showcasing the rich heritage and modern development of Diriyah as part of Saudi Vision 2030.

Course: CT1452 - Internet Applications

Instructor: Dr. Monerah Alawadh

Semester: 1447 (1)

Project Objectives

Our mission is to digitally preserve and promote the cultural heritage of Diriyah while highlighting its ongoing development as a global cultural destination.

Key Goals:

• Document historical and cultural significance

• Showcase development progress

• Provide educational resources

• Enhance visitor experience through digital means

Technical Implementation

This website was built using modern web technologies to ensure optimal performance and user experience across all devices.

Technologies Used:

HTML5, CSS3, JavaScript

Future Enhancements

We plan to continuously improve this platform with additional features and content to better serve our audience.

Planned Features:

• Virtual tours of historical sites

• Interactive timelines

• Multilingual support

• Enhanced gallery with 360° views
